## Getting Started

First get API KEY from [TMDB](https://www.themoviedb.org/)

Create local config file `touch .env.local`

Insert API KEY into .env.local

```
API_KEY = xxx...
```

Run the development server:

```bash
npm i
# or
yarn dev
# or
pnpm dev
```
